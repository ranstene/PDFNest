How It Works

Open the metadata tool, select a PDF, fill in document properties such as title, author, subject, and keywords, then download an updated PDF copy.

  1. Select one PDF file.
  2. Enter the metadata fields you want to update.
  3. Download the new PDF with updated document information.

Common Use Cases

Clean up document properties before publishing, sharing, archiving, indexing, uploading to a document system, or organizing PDFs on your device.

Privacy

Your browser reads the selected PDF, writes metadata into a new copy, and creates a temporary download link. Refreshing or closing the page clears selected files and generated links.

Limitations

The tool updates common PDF metadata fields. It does not remove all hidden content, rewrite visible page text, or edit password-protected PDFs unless they are unlocked first.

Tips for Best Results

  • Use a short title that describes the document clearly.
  • Add author and subject fields when organizing files for work or school.
  • Avoid putting private information in metadata unless it is meant to be shared.

Before You Download

Review title, author, subject, and keywords before saving because metadata can be visible in PDF readers.

Privacy and Ads

The tool runs in your browser and keeps document processing separate from advertising areas. PDFNest does not send selected document contents to an ad server, and ad containers should stay away from upload, action, and download controls.

FAQ

Does the Edit PDF Metadata tool upload my file?

No. PDFNest edits PDF metadata locally in your browser and does not upload documents to a server.

Which PDF metadata fields can I edit?

You can edit the PDF title, author, subject, and keywords before downloading an updated copy.
